\subsection{Detailed Description}
The exception that is thrown when an attempt to load a class fails due to the absence of an entry method. 

This method is intended as a drop in replacement for the equivalent exception in .N\+ET. Unfortunately, .N\+ET Standard 1.\+6 and earlier do not contain this exception, even though P/\+Invoke on Windows certainly raises this exception.
